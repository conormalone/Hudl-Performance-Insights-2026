# Hudl Performance Insights 2026

Soccer analytics research project exploring performance insights through data.

## Structure

```
roles/     — Role definitions for coding subagents
specs/     — Task briefs and specifications
code/      — Implementation output from subagents
outputs/   — Plots, tables, and results
assets/    — Datasets and reference data
```

## Workflow

1. Specs written in `specs/`
2. Subagents implement and test
3. Results land in `outputs/`
